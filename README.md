# Tic-Tac-Toe

## Overview
A simple two-player Tic-Tac-Toe game built using Python and Tkinter. This was one of my first programming projects and helped me understand GUI development, event handling, and basic game logic.

## Features
- 3x3 interactive game board
- Two-player mode: X and O
- Win detection for rows, columns, and diagonals
- Draw detection
- Automatic game reset after win or draw
- Simple Tkinter-based GUI

## Technologies Used
- Python
- Tkinter

## How to Run
1. Make sure Python is installed.
2. Clone the repository or download the code.
3. Run the Python file:

```bash
python tic_tac_toe.py
